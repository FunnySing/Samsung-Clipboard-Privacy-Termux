A serious concern has been raised by many that even if you delete clipboard items from its keyboard's clipboard section in Samsung, it still keeps latest 40 items cached in its deep system-level integration. And you feel fucked when you realize that some sensitive informations (password, email etc.) are also stored among them.

## About this script
This bash script will help copying 50 random numbers to your clipboard so that the old cached ones get vanished away.

## Requirements
1. [Termux](https://f-droid.org/packages/com.termux)
2. [Termux-api](https://f-droid.org/packages/com.termux.api)
